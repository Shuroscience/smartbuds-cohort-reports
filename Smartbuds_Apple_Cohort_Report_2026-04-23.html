What Was New in Apple

New Features

  • Adaptive Audio (Alpha) — real-time audio system that responds to sleep state, gated behind Labs toggle
  • Surveys & NPS — new survey system with NPS, insight kit surveys at Day 7 and Day 15
  • Day 2/4 Experience & Survey flows
  • Feature Intros — guided onboarding for ear adjustment, comfort check-in, and tips/wings selection
  • Signal Quality Tracking — EEG signal quality monitoring with anomaly detection
  • Tutorial Video Streaming — migrated from bundled assets to Firebase Storage, reducing app size
  • Neuro Tetris demo added to Labs
  • Onboard Audio During Deep Sleep — audio stimulation now supported during deep sleep phase
  • ZenDesk Support — migrated support links from Intercom to ZenDesk

Stability & Bug Fixes

  • Fixed main thread deadlock in CoreData; reduced SwiftUI re-render pressure
  • Improved CPU usage in sleep staging algorithm pipeline
  • Fixed alarm reliability (switched to DispatchSourceTimer)
  • Fixed Live Activity recovery and state caching
  • Numerous BLE reliability fixes — reconnection, RACE command race conditions, streaming restore
  • Fixed UI hangs during cloud sync; blocked sleep stats fetch during sync
  • Fixed session restore edge cases (streaming mode, sleep phase, app kill)
  • Fixed audio issues — ducking during alarms, soundscape resume, adaptive audio pausing

Infrastructure

  • Dependency injection and unit tests across core ViewModels and managers
  • Simulator target for running tests without hardware
  • CI pipeline on self-hosted ARM64 runner with scoped SwiftLint
  • AI-assisted Linear ticket triage pipeline
